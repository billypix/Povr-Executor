# Povr-Executor
I made a roblox executor that is server sided PLAY IN YOUR OWN GAME.
I'll soon try to make it so you can use it in a backdoored game.

## Setup
1. Insert a `RemoteEvent` into `ReplicatedStorage`
2. Put `ServerScript.lua` in `ServerScriptService`
3. Put `LocalScript.lua` inside the Execute button in your GUI

## Features
- Draggable UI
- Server-side execution via RemoteEvent
- Error handling with pcall
